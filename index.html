Oque e cliente no linguajar web

O Conceito de cliente e basicamente o consumidor aquele que vai utilizar o site apos sua criação. A internet e basicamente quem faz a ponte entre o seu computador e o servidor em que o site esta hospedado, quando o cliente clica em link para abrir um site a infromação e mandada atravez da internet para o servidor que por sua vez manda novamente ao cliente com sua velocidade de download podendo variar dependendo da velocidade de conexão do cliente e o tamanho e tipo de arquivo que vai ser exibido na tela do cliente. Afim de evitar um gigantesco trafego de informaçoes desnecessarias o seu navegador salva algumas informaçoes no seu computador gerando o arquivo de cache para inves de ele buscar novamente na internet o servidor em que o site esta hospedado ja utiliza o arquivo que esta salvo em seu computador para abrir o site e evitar esse trafego desnecessario de informaçoes.E importante no desenvolvimento de sites um design responsivo para que o site se adpte da melhor forma para a plataforma que o cliente esta utilizando seja ela um descktop ou um mobile tambem tendo uma melhor adaptabilidade em diferentes navegadores pois nem todos leem os codigos da mesma maneira.

Navegadores

Os navegadores são programas criados por empresas para abrir ou executar arquivos, os navegadores são utilizados para interpretar arquivos tais como Html, css, Javascrip, imagens e etc. os navegadores estão cada vez maiores agregando cada vez mais funcionalidades para abrir os mais diversos tipos de arquivos. Podem haver muitos navegadores de diversas empresas mas todos seguem o padrão criado pela w3c entretanto esses navegadores sempre tem uma difereça de interpretação dos mesmos arquivos.

Aplicações web

Aplicações web são basicamente algumas soluções criadas que tem a internet como meio de comunicação entre o cliente e o servidor não tendo necessidade de intalar elas na maquina do cliente. Temos diversas aplicações web como youtube e spootfy que tem tanto sua aplicação quando a sua versão para instalarmos na maquina do cliente.

Dispositivos moveis

Os dispositivos moveis hoje são uma parte importantissima do plublico em geral e tambem em algumas empresas programar para um dispositivo movel e diferente de programar para web são dois layouts diferentes mas ainda sim proximos logo sendo importante que tenha uma boa adaptabilidade boa em ambas as plataformas.

Servidores

Basicamente temos 2 tipos de servidores, locais são os que conectão uma rede local e os não locais que utilizam a internet para se conectar com a sua maquina. uma maquina de um cliente pode ser um servidor em uma rede local. Os servidores necessitam de uma maquina mais potente pois exigem um alto processamento de informaçoes e arquivos temos servidores de diversos tipos como os arquivos, segurança, streaming, e-mail, web e proxy.

Uma maquina pode ser mais de 1 servidor ao mesmo tempo mas isso poderia reduzir a sua eficacia tendo em vista que ela estaria divdindo seu processamento entre ambas funcionalidades, por isso e preferivel q um servidor seja apenas de 1 tipo como exemplo um de arquivos dedicando 100% de seu processamento para tal doque um de arquivos e straming q teria que dividir seu processamento entre os 2 podendo resultar em uma lentidão no procesamento de de arquivos e streaming.

Linguagem de Programação

Linguagem de programação é uma linguagem escrita e formal que especifica um conjunto de informações e regra para se gerar um software.Existem varios tipos de linguagem de programação alguns exemplos são Javascript, python, PHP, C# e etc. Dependendo da função que vai ser utilizada algumas são melhores que outras em alguns aspectos para as funçoes especificas como por exemplo para processamento de dados o Python e melhor que o java nesse aspecto mas sempre denpendo do contexto. A linguagem de progamção por si so não vai revolver os rpblemas que tiver um determinado software ela vai ser a ferramenta que vai te possibilitar resvolver esse problema seja ele um erro de programação ou uma formatação de texto para deixar o visual do site mais bonito.

Linguagem server-side e a linguagem que roda no lado do servidor por exemplo php e C#. pode ate ser a mesma informação mas se ela for em php vai ser ter uma forma de programar ou resvolver um problema ja em C# tem uma forma diferente de programar ou resolver o mesmo problema com a mesma informação.

Linguagem Client-side e uma linguagem que e rodada na maquina do cliente como exemplo Javascript os navegadores utilizam o V8 para interpretar a linguagem de Javascript no client-side tendo alguns recursos que voce não encontra no server-side e vice-versa.

Html na pratica

(Voltar)

No inicio de todo site temos html entre o sinal de < e o sinal de > gerando uma tag apos criar essa tag de html tudo entre o inicio e o fim dessa tag sera interpretado pelo navegador como um html.Utilizando entre a tag de html utilizamos outras duas tags entre eles que seriam o head e o body entre os sinais <> para dar forma ao site em html entre o head ficara aquilo que vai ser exibido na guia do navegador como exemplo (< head >Meu primeiro site < / head >), apos isso usamos a tag < body > < / body > tudo entre essa tag sera visto como o corpo do site.

Apos a colocarmos o body temos outras tags para dar forma ao texto temos tambem atributos gerados para tags para ajudar o html ser lido outras linguagens de programação.

W3schools neste site voce vera algumas tags e atributos para utilizar.
